تصنيف الـ APIs الخاصة بالملاك (Owners) في نظام غازي
==================================================

1. الـ APIs المشتركة لجميع الملاك (Shared APIs)
-----------------------------------------------
هذه الروابط يمكن لجميع أنواع الملاك الوصول إليها، وتتغير البيانات المعادة بناءً على دور المستخدم:

- GET  /api/owner/overview       : عرض نظرة عامة على النشاط التجاري (محطات، مغاسل، أو صيانة).
- GET  /api/owner/businesses     : قائمة المنشآت التابعة للمالك.
- GET  /api/owner/my-employees   : عرض قائمة جميع الموظفين التابعين للمالك في جميع منشآته.
- POST /api/owner/employees      : إضافة موظف جديد (يتم تحديد دوره تلقائياً بناءً على نوع المنشأة).
- GET  /api/owner/reports/revenue: تقارير الإيرادات (حالياً تدعم المحطات بشكل أساسي).

2. مالك محطة وقود (Station Owner)
----------------------------------
- GET  /api/owner/my-stations                 : قائمة المحطات المملوكة فقط.
- PUT  /api/owner/stations/{id}               : تحديث بيانات محطة معينة.
- GET  /api/owner/stations/{id}/today-refuels : إحصائيات التعبئة لمحطة معينة خلال اليوم.

3. مالك مغسلة سيارات (Car Wash Owner)
-------------------------------------
- GET  /api/owner/my-car-washes               : قائمة مراكز الغسيل المملوكة فقط.
- GET  /api/owner/car-washes/{id}/today-washes: إحصائيات عمليات الغسيل لمركز معين خلال اليوم.

4. مالك مركز صيانة (Maintenance Owner)
--------------------------------------
- GET  /api/owner/my-maintenance-centers       : قائمة مراكز الصيانة المملوكة فقط.
- GET  /api/owner/maintenance/{id}/today-services: إحصائيات خدمات الصيانة لمركز معين خلال اليوم.

ملاحظات تقنية:
- جميع هذه الروابط تتطلب تسجيل الدخول (Bearer Token).
- النظام يستخدم "Middleware" للتحقق من أن المستخدم يملك المنشأة التي يحاول الوصول لبياناتها.
- تم برمجة هذه المنطق في المتحكم: App\Http\Controllers\Api\OwnerController
